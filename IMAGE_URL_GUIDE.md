# Image URL Guide for Mini CMS

## How to Use Image URLs

Your CMS now supports image URLs from anywhere on the internet! Simply paste the direct image URL into the "Image URL" field when creating or editing posts.

## üìù How to Get Image URLs

### Method 1: Right-Click (Recommended)
1. Find an image you like on any website
2. Right-click on the image
3. Select "Copy image address" or "Copy image link"
4. Paste into the Image URL field in your CMS

### Method 2: Open in New Tab
1. Right-click on an image
2. Select "Open image in new tab"
3. Copy the URL from the browser address bar
4. Paste into your CMS

## ‚ö†Ô∏è Important Notes

- **Use HTTPS URLs**: Always use `https://` instead of `http://` for better security
- **Direct Image Links**: Make sure the URL ends with an image extension (.jpg, .png, .gif, .webp)
- **CORS Issues**: Some websites block their images from being embedded on other sites
- **Image Size**: Use reasonably sized images (under 2MB) for faster loading

## üé® Recommended Image Sizes

- **Thumbnail**: 800x600 pixels
- **Featured Image**: 1200x800 pixels
- **Banner**: 1920x1080 pixels

## üîß Troubleshooting

**Image not showing?**
- Check if the URL is correct and accessible
- Make sure it's a direct image link (ends with .jpg, .png, etc.)
- Try opening the URL in a new browser tab to verify it works
- Some websites block external embedding - try a different source

**Preview not working?**
- The preview feature requires a valid HTTP/HTTPS URL
- Wait a moment for the image to load
- If it doesn't appear, the URL might be invalid or blocked
